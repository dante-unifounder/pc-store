# TechPC Store — Landing page con asesor IA

Landing page de venta de componentes de PC con chatbot inteligente usando **Vercel AI SDKv** y **Google Gemini 3.1 Pro Preview**.

## Stack
- Next.js 14 (App Router)
- Vercel AI SDK v4 (`ai` + `@ai-sdk/google`)
- Modelo: `gemini-3.1-pro-preview`
- Tailwind CSS v3

## Inicio rápido
```bash
npm install && npm run dev
```
